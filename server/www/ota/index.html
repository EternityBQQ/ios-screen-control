📋 Mac 上操作步骤

1 克隆仓库:
git clone git@gitee.com:zhangzheng521/ios-screen-control.git
2 安装 XcodeGen:
brew install xcodegen
3 用免费 Apple ID 登录 Xcode
(设置 → Accounts → 点 + 登录)
4 一键打包:
bash build_ios.sh
5 上传到服务器:
scp ScreenCapture/build/ipa/ScreenCapture.ipa user@server:~/ios-screen-control/server/www/ota/
